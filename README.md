# Simulation (Симуляция)

Консольная игра «Симуляция» с возможностью выбора языка.

## Возможности

* Выбор языка игры: **английский** / **русский**

## Игровой процесс

* 5 видов сущностей - Лиса, Заяц, Росток, Дерево, Гора
* Лисы охотятся за Зайцами
* Зайцы ищут и поедают Ростки
* Каждый новый ход могут появится любые из сущностей 

## Установка и запуск

1. Создайте виртуальное окружение и активируйте его:

**Linux/macOS:**

```bash
uv venv 
source .venv/bin/activate
```

**Windows (PowerShell):**

```powershell
uv venv
.venv\Scripts\Activate.ps1
```

2. Установите пакет в режиме разработки:

```bash
pip install . -e
```

3. Запустите игру:

```bash
uv run simulation
```

Теперь игру можно запускать так:
```bash
simulation
```
